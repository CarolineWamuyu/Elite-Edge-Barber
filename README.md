# 💈 Elite Edge Barber

A premium barbershop website designed to showcase luxury grooming services, modern styling, and seamless appointment booking.

🔗 Live Demo: https://carolinewamuyu.github.io/Elite-Edge-Barber/

---

## Overview

Elite Edge Barber is a modern, responsive website that represents a high-end barbershop experience. The platform highlights professional grooming services, showcases craftsmanship, and allows users to book appointments بسهولة.

The site emphasizes:
- Clean UI/UX
- Premium aesthetic design
- Smooth navigation
- Interactive booking experience

---

##  Features

**Hero Section**
  - Strong branding message: *"Where legends are made"*
  - Call-to-action buttons (Book / Discover)

 **Services Section**
  - Precision Craft (Haircuts)
  - Royal Shave (Traditional grooming)
  - Beard Architecture
  - Emperor Experience (Full package)

 **Gallery Section**
  - Displays different haircut styles (classic, modern, beard work)

 **Testimonials**
  - Client feedback section to build trust

   **Booking System UI**
  - Input fields for:
    - Name
    - Email
    - Phone
    - Service selection

  **Responsive Design**
  - Works across desktop, tablet, and mobile
